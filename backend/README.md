# SwiftRise Backend

## Getting Started

1. Install dependencies:
   npm install

2. Start the server in development mode (with auto-reload):
   npm run dev

3. Or start the server normally:
   npm start

The server will run on http://localhost:5000 by default.

## API Endpoints
- `/` — Health check
- `/api/users` — Example users endpoint
- `/api/cars` — Example cars endpoint
- `/api/bookings` — Example bookings endpoint

CORS is enabled for local frontend development.
